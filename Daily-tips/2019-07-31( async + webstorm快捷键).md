### async

>async函数返回一个Promise对象。函数执行时，一旦遇到await就会先返回，等到遇到异步操作完成，再接着执行函数内后面的语句。
>
>`async`函数内部`return`语句返回的值，会成为`then`方法回调函数的参数。
>
>```javascript
>const fn = async ()=>{
>        console.log(1);
>        let a = await fn1()
>        console.log(a);
>    	return 3
>};
>const fn1 = async ()=>{
>     return new Promise(resolve => {
>         setTimeout(()=>{
>             console.log(2);
>             resolve(2)
>         },2000)
>      })
>};
>fn().then(a => console.log(a))
>```
>
